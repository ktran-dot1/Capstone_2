# Capstone 2: Business analysis with Python
### EmporiUm Sales Analysis

**This project is about the analysis of EmporiUm, a student bookstore chain. Using Python to read, clean, and analyze data for this business. Additionally, I will display my use of NumPy, pandas, and matplotlib. My manager has tasked me to analyze the sales within Connecticut and New York. I will be finding the monthly total revenue, ranking stores based on performance, top customers, and the transaction and revenue by product category per month. To support my findings, I created plot charts using matplotlib.**
